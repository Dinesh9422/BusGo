# 🚌 BusGo — Full Stack Bus Reservation System

A production-ready bus ticket booking platform built with Django REST Framework + React JS + MySQL.

## Tech Stack
- Backend: Django 4.2, Django REST Framework, JWT Auth
- Frontend: React 18, React Router v6
- Database: MySQL

## Features
- 🔐 JWT Authentication
- 🔍 Bus Search with filters
- 💺 Visual Seat Selection
- 🎫 Booking with passenger details
- 💳 Dummy Payment Gateway
- 📄 PDF Ticket Download
- ❌ Booking Cancellation
- 🏆 Loyalty Points System
- 🎟️ Promo Code Discounts
- 🔔 Real-time Notifications
- ⭐ Bus Ratings & Reviews
- 📊 Trip Progress Tracker
- 🌙 Dark Mode
- 📱 Mobile Responsive
